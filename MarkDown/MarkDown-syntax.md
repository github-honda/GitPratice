Markdown 是一種輕量級的標記語言，常用於撰寫說明文件、README 檔案、部落格文章等。它的語法簡單直觀，可以快速轉換成 HTML。以下是常用的 Markdown 語法整理：

---

## 📌 標題（Headers）

使用 `#` 表示標題層級（1~6 級）：

```markdown
# 標題 1
## 標題 2
### 標題 3
#### 標題 4
##### 標題 5
###### 標題 6
```

---

## ✏️ 文字樣式

```markdown
**粗體** 或 __粗體__
*斜體* 或 _斜體_
~~刪除線~~
`程式碼`（單行）
```

測試:

連續文字 **粗體1** 或 __粗體2__ 或 *斜體1* 或 _斜體2_ 或 ~~刪除線~~ 或`單行程式碼` 最後在這裡. 

測試
下一句

## 📋 清單

### 無序清單：

```markdown
- 項目 A
* 項目 B
+ 項目 C
```

### 有序清單：

```markdown
1. 第一項
2. 第二項
3. 第三項
```

測試:
* 開始一個 *, 並且沒有結尾的 *, 會被視為項目.
* 如上.
* 如上.
- 混雜項目1
- 混雜項目2
+ 項目 C
+ + 項目 C

1. aaa
2. bbb
3. 1. ccc
3. 2. ddd
4. ddd

** 兩個星號沒影響

*** 三個星號沒影響

**** 四個星號沒影響

## 📦 區塊引用（Blockquote）

```markdown

```
